# lop_vpf02
Programa simples em C que lê um arquivo `vendas.csv`, calcula o total vendido por cada produto e mostra um relatório rápido no terminal.

##  Como Funciona

1. Abre o arquivo `vendas.csv`.
2. Pula a linha de cabeçalho.
3. Lê o nome, a quantidade e o preço de cada produto.
4. Calcula: `Total = Quantidade × Preço`.
5. Mostra os resultados organizados na tela.
